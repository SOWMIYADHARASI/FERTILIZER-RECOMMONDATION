# 🌱 AI Fertilizer Recommendation System

An AI-powered web application that helps farmers make better agricultural decisions by analyzing soil test reports and providing personalized fertilizer and crop recommendations using Google Gemini AI.

## 🚀 Live Demo

🔗 **Project Link:** https://fertilizer-recommondation.onrender.com

---

## 📌 Features

- 📄 Upload soil test reports (PDF/Image)
- 🔍 Automatically extract soil parameters
- 🤖 AI-powered fertilizer recommendations using Google Gemini AI
- 🌾 Suggest suitable crops based on soil conditions
- 📊 Display detailed soil analysis and recommendations
- 💬 Simple and user-friendly interface
- ⚡ Fast and accurate results

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/AI-Fertilizer-Recommendation.git
```

### 2. Navigate to the project

```bash
cd AI-Fertilizer-Recommendation
```

### 3. Install dependencies

```bash
npm install
```

### 4. Create a `.env` file

```env
GEMINI_API_KEY=YOUR_GOOGLE_GEMINI_API_KEY
```

### 5. Run the application

```bash
npm run dev
```

## 📖 How It Works

1. Upload a soil test report.
2. The system extracts soil parameters.
3. Google Gemini AI analyzes the extracted data.
4. The application recommends:
   - Suitable fertilizers
   - Suitable crops
   - Soil improvement suggestions
5. The results are displayed in a simple and easy-to-understand format.
